# Internal Ops Agent

Canonical internal-ops starter project generated by create-plico.

Start with `agent.md`, then inspect `skills/`, `tools/`, `evals/`, `memory/`, and `artifacts/`.

This example includes a safe declared `create_ticket.tool.ts` module and a matching `evals/smoke.script.json` so `plico run --dry --script examples/internal-ops-agent/evals/smoke.script.json examples/internal-ops-agent` can exercise the runtime smoke path locally.

Validate this project from the repo root with:

```sh
pnpm --filter @plico/cli exec plico validate examples/internal-ops-agent
```
